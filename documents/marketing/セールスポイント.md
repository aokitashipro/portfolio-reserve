## 💎 3つの核心的価値

### 1. コスト削減 - 年間56万円の節約

**現状の課題：**
```
ホットペッパー
├─ 月額基本料: 3〜10万円
├─ 予約手数料: 1件200〜500円
└─ 年間コスト: 約60万円（月間100件予約の場合）
```

**このシステム導入後：**
```
自社予約システム
├─ 月額費用: 約3千円（インフラ）
├─ 予約手数料: 0円
└─ 年間コスト: 約3.6万円
```

**削減額: 年間56.4万円 💰**

### 2. 顧客データの資産化

**ホットペッパーの問題：**
- ❌ 顧客データが蓄積されない
- ❌ 他店への誘導リスク
- ❌ 退会すると顧客を失う

**このシステムの強み：**
- ✅ 全ての顧客データを自社で完全保持
- ✅ 来店履歴・施術履歴を蓄積
- ✅ リピート率分析で改善施策を立案
- ✅ 休眠顧客の掘り起こし

**具体例：**
- 「前回パーマをかけた佐藤様、3ヶ月経過しました」→自動リマインド
- 「カラーリングの顧客リピート率が60%」→改善施策を検討
- 「誕生月の顧客に自動クーポン送信」→再来店促進

### 3. 店舗運営の効率化

**TimeRexや他ツールにない機能：**

| 機能 | 効果 |
|------|------|
| **スタッフ管理** | シフト最適化で人件費削減 |
| **メニュー分析** | 収益性の高いメニューを特定 |
| **予約状況の可視化** | 空き時間の有効活用 |
| **自動リマインド** | 無断キャンセル率の削減 |

## 🚀 よくある質問（FAQ）

### Q1. 技術的な知識がなくても使えますか？

**A.** はい、問題ありません。直感的な操作画面で、初めての方でも簡単に使えます。また、操作マニュアルと動画解説も提供します。

### Q2. スマホからも予約できますか？

**A.** はい、レスポンシブデザインでスマホ・タブレット・PCすべてに対応しています。顧客の約80%がスマホから予約すると想定して設計しています。

### Q3. 既存の顧客データを移行できますか？

**A.** はい、CSVファイルでの一括インポート機能をサポートします。既存データの移行もお手伝いします。

### Q4. デザインをカスタマイズできますか？

**A.** はい、オプション料金でブランドカラーやロゴの適用、完全オリジナルデザインにも対応します。

### Q5. 複数店舗で使えますか？

**A.** はい、マルチテナント設計で複数店舗に対応可能です。店舗ごとの料金についてはご相談ください。

### Q6. セキュリティは大丈夫ですか？

**A.** Supabase（エンタープライズグレード）を使用し、SSL通信・データ暗号化・定期バックアップで安全性を確保しています。

### Q7. 途中で解約できますか？

**A.** はい、いつでも解約可能です。ただし、初期費用の返金は行っておりません。

### Q8. 月額費用は本当に3千円だけですか？

**A.** インフラ費用（Vercel + Supabase）が約3千円です。月間予約件数が大幅に増えた場合は増額の可能性がありますが、基本的には定額です。
