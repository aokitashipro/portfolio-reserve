## ♿ アクセシビリティ

### WCAG 2.1 AA準拠

**実装ガイドライン**:

1. **カラーコントラスト**:
   - テキストとbackgroundのコントラスト比: 最低 4.5:1
   - 大きいテキスト（18pt以上）: 最低 3:1

2. **キーボード操作**:
   - すべてのインタラクティブ要素はキーボードでアクセス可能
   - Tabキーでのフォーカス順序が論理的
   - フォーカスインジケーター表示: `focus:ring-2 focus:ring-blue-500`

3. **スクリーンリーダー対応**:
   - セマンティックHTML使用（`<header>`, `<nav>`, `<main>`, `<footer>`）
   - `aria-label` でアイコンボタンに説明を追加
   - フォーム入力に `<label>` を必ず設定

4. **フォーカス管理**:
   ```tsx
   <button className="... focus:outline-none focus:ring-2 focus:ring-blue-500 focus:ring-offset-2">
     送信
   </button>
   ```

5. **代替テキスト**:
   - すべての画像に `alt` 属性
   - 装飾的な画像は `alt=""` で空にする

**新しいページ・コンポーネントを作成する際の確認事項**:

- [ ] カラーパレットに従っているか
- [ ] フォントサイズ・ウェイトは階層に沿っているか
- [ ] スペーシングは4pxベースで統一されているか
- [ ] レスポンシブデザインが正しく動作するか
- [ ] モバイルでタッチターゲットサイズが44px以上あるか
- [ ] キーボード操作が可能か
- [ ] フォーカスインジケーターが表示されるか
- [ ] カラーコントラスト比が4.5:1以上か
- [ ] 既存コンポーネントを再利用できないか
